
## 📦 Requirements

Make sure you have installed:

- Node.js (>= 18)
- npm or yarn
- Git

## 🔧 Installation

Clone the repository:

```bash
git clone https://github.com/sebasGZA/frontend
cd frontend
npm install
```

## ⚙️ Environment Variables

Create a .env.local file in the root directory and set the variables as .env-template file example

## ▶️ Running the Project
```bash
npm run dev
```

## ▶️ Running the tests
```bash
npm run test:watch
```
